# markw56127.github.io

A static GitHub Pages portfolio for `markw56127`.

## Local Preview

Open `index.html` in a browser, or run a tiny local server from this folder:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Customize

- Edit profile copy, skills, and contact links in `index.html`.
- Adjust colors and layout in `styles.css`.
- Change the GitHub username or project selection logic in `script.js`.
